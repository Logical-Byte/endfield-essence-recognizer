# 终末地基质妙妙小工具

不知道哪些基质该留哪些该扔？想要当狗粮又担心万一这个基质有用？快来逝逝终末地基质妙妙小工具罢！

## 使用前阅读

- 请使用**管理员权限**运行本工具，否则无法捕获全局热键
- 请在终末地的设置中将分辨率调整为 **1920×1080 窗口**
- 请按 `N` 键打开终末地**贵重品库**并切换到**武器基质**页面
- 在运行过程中，请确保终末地窗口**置于前台**

## 功能介绍

- 按 `[` 键识别当前选中的基质，仅识别不操作
- 按 `]` 键扫描所有基质，并自动锁定宝藏基质，解锁其他基质<br>
  基质扫描过程中再次按 `]` 键中断扫描
- 按 `Alt+Delete` 退出程序

**宝藏基质和养成材料：** 如果这个基质和任何一把武器能对上（基质的所有属性与至少 1 件已实装武器的属性完全相同），则是宝藏，否则是养成材料。

## 开发

### frontend

前端部分位于 `frontend` 目录。

安装 node.js 后，安装前端依赖：

```bash
cd frontend
npm install
```

### backend

后端位于 `src/endfield_essence_recognizer` 目录。

我们建议使用 `uv` 管理虚拟环境：

```bash
uv sync --group build
```

### 运行

要完整运行该工具，需要同时启动前后端：

```bash
cd frontend
npm run dev
```

```bash
uv run eer
```

环境变量在 `.env` 文件中配置。
